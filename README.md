## Installation
* `git clone https://github.com/srpsatish/react-virtual-rupees.git`
* `cd react-virtual-rupees`
* `yarn`
* `yarn start`
Runs the app in the development mode.<br /> 
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
